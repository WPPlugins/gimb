This plugin adds a new media button that can insert images directly from Google Images.

== Description ==

GIMB adds a new media button from where you can insert images directly from Google Images.

== Installation ==

1. Upload `gimb` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings -> GIMB Options and provide the necessary information
4. Go to Posts -> Add New and click 'G' button
5. Make your search and click 'include'
